Life Expectancy Regression Analysis

A statistical analysis exploring the relationship between life expectancy and various socio-economic factors using multiple linear regression.

Tools Used:

  • R
  • RStudio
View on GitHub

Project Overview

This is a final project from the Regression Analysis course at IPB University that investigates how various socio-economic variables affect life expectancy using multiple linear regression. The goal was to build a reliable model, evaluate its assumptions, and extract meaningful insights that could support public health strategies.

In this project, I learned how to:

  1. Perform Exploratory Data Analysis (EDA) to clean, transform, and visualize datasets.
  2. Build, refine, and interpret multiple linear regression models with backward selection.
  3. Apply hypothesis testing (t-tests, F-tests) to evaluate model significance and reliability.
  4. Diagnose and address key regression assumptions such as multicollinearity, homoskedasticity, normality of residuals, and autocorrelation.
  5. Use R to implement statistical modeling, run diagnostics, and create clear visualizations.

Solaria Personalized Menu Recommendation System

A personalized menu recommendation system for Solaria based on user ratings using machine learning.

Tools Used:

  • Julia
  • SVD
View on GitHub View Paper

Project Overview

This is a group project from the Mathematical Computing Applications course at IPB University that develops a personalized menu recommendation system for Solaria restaurants using machine learning techniques, specifically Singular Value Decomposition (SVD). The goal was to leverage customer rating data to generate personalized menu suggestions and improve customer satisfaction.

Highlights:

  1. Designed and conducted a survey through Google Form to collect customer rating data.
  2. Preprocessed the dataset and construct a user–item matrix to represent menu preferences using Julia.
  3. Applied Singular Value Decomposition (SVD) for matrix factorization to fill in missing ratings and generate recommendations.
  4. Performed hyperparameter tuning to select the best number of latent factors for accurate predictions.
  5. Evaluated recommendation model performance.
